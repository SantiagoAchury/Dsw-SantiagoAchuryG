Proposito

Flex Box y CSS Grid Este sitio web tiene como proposito poder identificar y diferenciar estos dos tipos de elementos como el Flex box y el Css Grid, los cuales contienen una propiedad cada uno y nos indican la capacidad de alterar los diseños de dimensiones en una columna o varias.

Flex box

"La propiedad flexbox es una propiedad resumida que indica la capacidad de un elemento flexible para alterar sus dimensiones y llenar el espacio disponible"

CSS Grid
